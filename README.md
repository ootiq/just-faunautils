# justfaunautils

Just helpers and utilities for the `FaunaDB Javascript Driver`

## Install

```
yarn add @ootiq/justfaunautils
```

## Included

### Functions

-   **getClient**
-   **q**
-   **auth/FaunaLogin**
-   **auth/TokenLogout**
-   **auth/AuthToken**
-   **auth/AuthTokenByIndex**
-   **query/MatchIndex**
-   **query/MatchCollections**
-   **query/CreateData**

### Types

-   **FaunaResponseProps**
